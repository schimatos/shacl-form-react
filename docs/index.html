shacl-form-react

SHACL Form React

A HTML5 form generator for any SHACL constraint compliant to the W3C SHACL standard.

Usage

 

export function MyApp() {
  return <SHACLForm
      shacl={shacl}
      data={data}

    />
}

This library also exposes the SHACLFormInternal component.
